Enum Lifetime

Describes the lifetime of a service. A service can be registered in various ways. For more information check out the IServiceContainerExtensions class.

Namespace: Solti.Utils.DI
Assembly: Injector.dll
Syntax
public enum Lifetime

Fields

Name Description
Scoped

Services having Scoped liftime are instantiated only once (per injector) on the first request and disposed automatically when the parent injector is disposed.
Singleton

Services having Singleton liftime are instantiated only once (per declaring container) on the first request and disposed automatically when the declaring container is disposed.
Transient

Services having Transient lifetime are instantiated on every request. The caller must dispose them (if necessary).
