Registers a new service factory with the given type. Factories are also services except that the instantiating process is delegated to the caller. Useful if a service has more than one constructor:

ScopeFactory.Create
(
    svcs => svcs.Factory(typeof(IMyService), (scope, iface) => new MyService(...), Lifetime.Singleton),
    ...
)
Declaration
public static IServiceCollection Factory(this IServiceCollection self, Type iface, Expression<FactoryDelegate> factory, LifetimeBase lifetime, ServiceOptions options = null)
Parameters
Type Name Description
IServiceCollection self

The target IServiceCollection.
Type iface

The service interface to be registered. It can not be null and can be registered only once.
Expression<FactoryDelegate> factory

The factory function that is responsible for the instantiation. Its call count depends on the value of the lifetime parameter. Note that the second parameter of the factory is never generic, even if you registered the factory for an open generic interface.
LifetimeBase lifetime

The lifetime of service.
ServiceOptions options

Options to be assigned to the service being registered.
Returns
Type Description
IServiceCollection
Remarks

You can register generic services (where the iface parameter is an open generic type).

The service implementation to be registered. It can not be null and must implement the iface interface. Additionally it must have only null or one constructor (that may request another dependecies).
