Enum Lifetime

Describes the lifetime of a service.

Namespace: Solti.Utils.DI.Interfaces
Assembly: Solti.Utils.DI.Interfaces.dll
Syntax
public enum Lifetime

Fields

Name Description
Scoped

Services having Scoped liftime are instantiated only once (per IInjector) on the first request and disposed automatically when the parent IInjector is disposed.
Singleton

Services having Singleton liftime are instantiated only once (in declaring IServiceContainer) on the first request and disposed automatically when the container is disposed.
Transient

Services having Transient lifetime are instantiated on every request and released automatically when the parent IInjector is disposed.
