Pooled

Services having pooled lifetime are instantiated in a separate pool. Every IInjector may request a service instance from the pool which is automatically resetted and returned when the parent IInjector is disposed.

Declaration
public static Lifetime Pooled { get; protected set; }

Remarks

Pooled services should implement the IResettable interface.

Scoped

Services having scoped liftime are instantiated only once (per IInjector) on the first request and disposed automatically when the parent IInjector is disposed.
