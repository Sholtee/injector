Proxy(IServiceContainer, Type, String, Type)

Hooks into the instantiating process to let you decorate the original service. Useful when you want to add additional functionality (e.g. parameter validation). The easiest way to decorate an instance is using the InterfaceInterceptor<TInterface> class.
