Gets the IServiceContainer associated with the injector.

Declaration
IServiceContainer UnderlyingContainer { get; }
Property Value
Type Description
IServiceContainer
Remarks

Every injector has its own service container that serves it on service request. This container is a direct descendant of the container from which the injector was created.
