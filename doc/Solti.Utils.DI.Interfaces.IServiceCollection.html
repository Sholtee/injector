Interface IServiceCollection

Specifies the contract for an AbstractServiceEntry set.

Namespace: Solti.Utils.DI.Interfaces
Assembly: Solti.Utils.DI.Interfaces.dll

Remarks

IServiceCollection interface implements the ISet<T> interface so you can register only one entry with the given Interface and Name.
